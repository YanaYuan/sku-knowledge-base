SKU 解剖图

每个 Item Name 都遵循固定的 token 结构。下面是一个典型 Azure VM SKU 的完整拆解：

Az VM Ev3/ESv3 Srs - E32 v3/E32s v3 - 10 Hrs - AU E
Platform = Azure Type = VM Series = E v3 Size = 32 vCPU Qty = 10 Unit = Hours Region = AU East

General Pattern

[Platform] [Service] [Series/Version] Srs [Win]-[Size]-[Qty] [Unit]-[Modifier]-[Region]

Modifiers (optional): DvTst = Dev/Test | Low Pri = Low Priority | Win = Windows License
