SKU 解剖图

每个 Item Name 都遵循固定的 token 结构。下面是一个典型 Azure VM SKU 的完整拆解：

Az VM Ev3/ESv3 Srs - E32 v3/E32s v3 - 10 Hrs - AU E
平台 = Azure 类型 = Virtual Machine 系列 = E-series v3 标签 = Series 规格 = 32 vCPU 数量 = 10 单位 = Hours 区域 = Australia East

通用模式：

[平台] [服务类型] [系列/版本] Srs [Win]-[规格]-[数量] [单位]-[修饰符]-[区域]

修饰符 (可选): DvTst = 开发测试 | Low Pri = 低优先级 | Win = Windows 许可
